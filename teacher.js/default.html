知识点一:人类的出现

例1 下列说法错误的是 ( )

A.人类形成于距今约三四百万年前

B.南方古猿是世界上最早的人类

C.由于自然地理环境的不同，人种的差异显现了出来

D.直立行走是人类进化史上具有决定意义的一步

实际上，南方古猿还属于"正在形成中的人"， 还不能算作真正意义上的人类。它经过浸长的发展阶 段，发展到"完全形成的人"的时候，人类才最终与猿类 分道汤锅，人类才真正形成。所以 B 选项的说法是错 误的。

【答案】 B

知识点二:氏族公社

例2 促使原始社会由母系氏族社会向系氏族社会过渡的根本原因是 ( )

A.妇女在氏族中地位的变化

B.生产力水平的提高

c.婚姻关系的变革

D.男子在氏族中地位的提高

本题的四个选项看起来好像都有道理，但 A c、D 三个选项都是母系氏族社会向父系氏族社会过渡过程中的表面现象，而所有这一切变化的根本原因都 是生产力水平的提高。经济基础决定上层建筑，生产 力决定生产关系，人类社会的发展都是以生产力水平 的提高为基础的。所以只有B项才是母系氏族社会向 父系氏族社会过渡的根本原因

【答案】 B
